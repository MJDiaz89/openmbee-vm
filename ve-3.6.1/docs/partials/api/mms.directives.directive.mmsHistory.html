mmsHistory
directive in module mms.directives

Description

Outputs a history window of the element whose id is specified. History includes name of modifier and date of change. Also modified date links to spec output below.

template (html)

Example for showing an element history

    <mms-history mms-element-id="{{id}}" mms-ref-id="{{refId}}"
                         mms-project-id="{{projectId}}"></mms-history>

Dependencies

  • ElementService
  • ProjectService
  • $templateCache
  • $q
  • _

Usage

as element:
<mms-history
       mms-element-id="{string}"
       mms-project-id="{string}"
       mms-ref-id="{string=master}">
</mms-history>

Parameters

ParamTypeDetails
mmsElementIdstring

The id of the element

mmsProjectIdstring

The project id for the element

mmsRefIdstring=master

Reference to use, defaults to master

Methods

  • changeElement()
