mmsJobs
directive in module mms.directives

Description

Shows you jobs that you have running, going to run and history of jobs. As well as let you edit and add jobs

template (html)

Example for showing an element jobs

  <mms-jobs></mms-jobs>

Dependencies

  • $templateCache
  • $http
  • $location
  • _

Usage

as element:
<mms-jobs
       mms-branch="{string=master}"
       mms-doc-id="{string=null}">
</mms-jobs>

Parameters

ParamTypeDetails
mmsBranchstring=master

Branch to use, defaults to master

mmsDocIdstring=null

the id of the current document under which the job is being run
