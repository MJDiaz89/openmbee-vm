mmsDiffAttr
directive in module mms.directives

Description

Compares a element at two different refs/commits and generates a pretty diff.

Example

<mms-diff-attr mms-base-element-id="" mms-attr="name|doc|val" (mms-base-project-id="" mms-base-ref-id="" mms-compare-ref-id="" mms-base-commit-id="" mms-compare-commit-id="")>

Dependencies

  • ElementService
  • $compile
  • $rootScope
  • $interval

Usage

as element:
<mms-diff-attr
       mms-attr="{string}"
       mms-base-element-id="{string}"
       mms-base-project-id="{string}"
       mms-compare-project-id="{string}"
       mms-base-ref-id="{string=master}"
       mms-compare-ref-id="{string=master}"
       mms-base-commit-id="{string=latest}"
       mms-compare-commit-id="{string=latest}">
</mms-diff-attr>

Parameters

ParamTypeDetails
mmsAttrstring

Attribute to use - ie name, doc or value

mmsBaseElementIdstring

The id of the element to do comparison of

mmsBaseProjectIdstring

Base project ID for original/base element

mmsCompareProjectIdstring

Compare project ID for compare element

mmsBaseRefIdstring=master

Base ref ID or master, defaults to current ref or master

mmsCompareRefIdstring=master

Compare ref ID or master, defaults to base ref ID

mmsBaseCommitIdstring=latest

Base commit id, default is latest

mmsCompareCommitIdstring=latest

Compare commit id, default is latest
