mmsRefList
directive in module mms.directives

Description

Displays a list of branches/tags with details. Provides options for taking action on ref. For the time being it only allows for running a doc merge job on current document.

Dependencies

  • $templateCache
  • $http
  • $location
  • _

Usage

as element:
<mms-ref-list
       mms-project-id="{string=master}"
       mms-ref-id="{string=master}"
       mms-doc-id="{string=null}">
</mms-ref-list>

Parameters

ParamTypeDetails
mmsProjectIdstring=master

Current project

mmsRefIdstring=master

Current branch

mmsDocIdstring=null

the id of the current document
