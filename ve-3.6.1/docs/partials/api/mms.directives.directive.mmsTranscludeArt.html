mmsTranscludeArt
directive in module mms.directives

Description

Given an artifact id, puts in an img or link to artifact.

Dependencies

  • ArtifactService
  • AuthService
  • ElementService
  • URLService
  • growl

Usage

as element:
<mms-transclude-art
       mms-element-id="{string}"
       mms-project-id="{string}"
       mms-ref-id="{string=master}"
       mms-commit-id="{string=latest}">
</mms-transclude-art>

Parameters

ParamTypeDetails
mmsElementIdstring

The id of the artifact

mmsProjectIdstring

The project id

mmsRefIdstring=master

Reference to use, defaults to master

mmsCommitIdstring=latest

Commit ID, default is latest
