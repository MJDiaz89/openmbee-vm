mmsCf
directive in module mms.directives

Description

Given an element id, puts in the element's name binding, if there's a parent mmsView directive, will notify parent view of transclusion on init and name change, and on click

Dependencies

  • $compile

Usage

as element:
<mms-cf
       mms-element-id="{string}"
       mms-project-id="{string}"
       mms-cf-type="{string}"
       mms-ref-id="{string=master}"
       mms-commit-id="{string=latest}"
       non-editable="{boolean=false}"
       mms-watch-id="{bool}">
</mms-cf>

Parameters

ParamTypeDetails
mmsElementIdstring

The id of the view

mmsProjectIdstring

The project id for the view

mmsCfTypestring

one of doc, val, name, com, img

mmsRefIdstring=master

Reference to use, defaults to master

mmsCommitIdstring=latest

Commit ID, default is latest

nonEditableboolean=false

can edit inline or not

mmsWatchIdbool

set to true to not destroy element ID watcher
