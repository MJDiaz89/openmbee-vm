mmsViewPresentationElem
directive in module mms.directives

Description

Given a InstanceVal, parses the element reference tree to get the corresponding presentation element, and renders it in the view

Dependencies

  • ViewService
  • ElementService
  • $templateCache
  • $location
  • $timeout
  • $rootScope
  • $anchorScroll
  • growl

Usage

as element:
<mms-view-presentation-elem
       mms-instance-val="{Object}"
       mms-parent-section="{Object}">
</mms-view-presentation-elem>

Parameters

ParamTypeDetails
mmsInstanceValObject

A InstanceValue json object

mmsParentSectionObject

the parent section if available
