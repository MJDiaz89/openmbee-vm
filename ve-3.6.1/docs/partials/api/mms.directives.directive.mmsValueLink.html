mmsValueLink
directive in module mms.directives

Description

Given an element id, generates a hyperlink with a cross-reference value

Dependencies

  • ElementService

Usage

as element:
<mms-value-link
       mms-element-id="{string}"
       mms-project-id="{string}"
       mms-ref-id="{string=master}"
       mms-commit-id="{string=latest}"
       mms-error-text="{string}"
       mms-link-text="{string}">
</mms-value-link>

Parameters

ParamTypeDetails
mmsElementIdstring

The id of the view

mmsProjectIdstring

The project id for the view

mmsRefIdstring=master

Reference to use, defaults to master

mmsCommitIdstring=latest

Commit ID, default is latest

mmsErrorTextstring

Text to display when element is not found

mmsLinkTextstring

Text to display for hyperlink
