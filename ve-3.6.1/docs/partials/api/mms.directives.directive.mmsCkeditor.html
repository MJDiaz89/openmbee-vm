mmsCkeditor
directive in module mms.directives

Description

Make any textarea with an ngModel attached to be a CKEditor wysiwyg editor. This requires the CKEditor library. Transclusion is supported. ngModel is required.

Example
   <textarea mms-ckeditor ng-model="element.documentation"></textarea>

Dependencies

  • CacheService
  • ElementService
  • UtilsService
  • ViewService
  • $uibModal
  • $templateCache
  • $window
  • $timeout
  • growl
  • CKEDITOR
  • _

Usage

as attribute
<textarea mms-ckeditor>
   ...
</textarea>
