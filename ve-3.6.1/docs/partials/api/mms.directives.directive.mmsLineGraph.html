mmsLineGraph
directive in module mms.directives

Description

A directive for generating a D3 line graph from opaque tables. Element text is used as caption. If element text is "$title", the table title will be used instead. Use "{''}" to generate an empty caption.

Usage

as element:
<mms-line-graph
       eid="{string}"
       [series-names="{string[]}"]
       [x-col="{string}"]
       [x-cols="{string[]}"]
       [y-col="{string}"]
       [y-cols="{string[]}"]
       [x-label="{string}"]
       [x-label-pos="{string}"]
       [y-label="{string}"]
       [y-label-pos="{string}"]
       [log-scale="{string|number}"]
       [log-scale-x="{string|number}"]
       [log-scale-x="{string|number}"]
       log-scale-label="{boolean=true}"
       [tick-fit="{string}"]
       type="{string="line"}"
       [padding="{object}"]>
</mms-line-graph>
as attribute
<ANY mms-line-graph
     eid="{string}"
     [series-names="{string[]}"]
     [x-col="{string}"]
     [x-cols="{string[]}"]
     [y-col="{string}"]
     [y-cols="{string[]}"]
     [x-label="{string}"]
     [x-label-pos="{string}"]
     [y-label="{string}"]
     [y-label-pos="{string}"]
     [log-scale="{string|number}"]
     [log-scale-x="{string|number}"]
     [log-scale-x="{string|number}"]
     log-scale-label="{boolean=true}"
     [tick-fit="{string}"]
     type="{string="line"}"
     [padding="{object}"]>
   ...
</ANY>

Directive info

  • This directive creates new scope.

Example

Example usage with multiple single-series tables. 

// Generates a velocity-over-time graph with 4 time series and a caption
// '###' are the element IDs of tables with columns:
// "Acceleration (m/s^2), "Time (s)", "Velocity (m/s)"
// and:
// "Acceleration (m/s^2), "Time", "Velocity (m/s)"
<mms-line-graph data-eid="###,###,###,###" data-y-col="Velocity" data-x-cols="['Time (s)', 'Time']">
 Caption this!
</mms-line-graph>
