mmsGroupDocs
directive in module mms.directives

Description

Dependencies

  • ElementService

Usage

as element:
<mms-group-docs
       mms-group-id="{string}"
       mms-ref-id="{string=master}"
       mms-project-id="{string}">
</mms-group-docs>

Parameters

ParamTypeDetails
mmsGroupIdstring

The id of the group to show documents for

mmsRefIdstring=master

Ref, defaults to master

mmsProjectIdstring

Project Id, if not stated will get from surrounding view
